# 需求规格说明书

## 1. 项目概述

### 1.1 项目背景
- **项目名称**：SmartCheck AI - 食堂后厨晨检系统
- **目标用户**：学校食堂后厨员工
- **核心目标**：检查食堂后厨员工的工作情况是否合格，确保从业人员健康上岗

### 1.2 产品定位
- 面向学校食堂的后厨员工晨检系统
- 无人值守自助式操作
- 嵌入式部署在工业主板上

### 2.2 首页（ Dashboard）

| 功能 | 描述 |

### 2.3 我要晨检模块

| 功能 | 描述 |
|------|------|
| 人脸识别 | 实时摄像头识别员工身份 |
| 信息展示 | 识别成功后显示：员工照片、姓名、体温、健康证有效期 |
| 手部检测 | 识别手心、手背，检测是否有异物 |
| 照片记录 | 记录人脸照片、手心、手背照片 |
| 异常提示 | 检测到异物时显示提示 |
| 提交 | 完成晨检后提交记录 |

#### 晨检流程：
```
人脸识别 → 显示员工信息 → 体温测量 → 手心检测 → 手背检测 → 提交
```

### 2.8 业务规则

#### 健康证有效期检查
| 状态 | 处理方式 |
|------|----------|
| 正常（>7天） | 正常晨检 |
| 即将过期（≤7天） | 显示警告提示，但允许继续 |
| 已过期 | 禁止上岗，弹出"健康证已过期"提示，阻止晨检 |

#### 晨检结果判定
| 检查项 | 判定标准 |
|--------|----------|
| 体温 | ≤ 37.5°C 为正常，> 37.5°C 为异常 |
| 手部检测 | 未检测到异物为正常，检测到异物为异常 |
| 健康证 | 未过期为正常，已过期禁止上岗 |

## 6. 手部异物检测

### 异物检测类型（技术实现）
| 英文 | 中文 |
|------|------|
| band_aid | 创可贴/绷带 |
| bracelet | 手镯 |
| ring | 戒指 |
| watch | 手表 |
| toilet_paper | 纸巾 |
| foreign | 其他异物 |
| hand | 手 |

### 检测规则
- 手心/手背顺序是规定死的（先手心后手背）
- 检测异物时用框框住手部，若检测到异物也框住异物
- 手部识别有异物 → 弹出警告 → 不能进入下一步 → 需重测

---

## 7. 晨检步骤说明

### 完整晨检流程
```
1. 点击"我要晨检"进入
2. 人脸检测 → 识别成功后显示姓名、健康证到期天数
3. 切换手部摄像头 → 手心检测 → 记录照片（有框）
4. 翻转手背 → 手背检测 → 记录照片（有框）
5. 判定：
   - 若有异物 → 弹出警告 → 不能进入下一步（可重测手部）
6. 询问"今天有没有不舒服？"
   - 有不舒服 → 不通过 → 重新检测
   - 无不舒服 → 通过 → 保存记录 
```

### 各步骤可重新拍摄
- 人脸识别：可点击重新拍摄
- 手心识别：可点击重新拍摄
- 手背识别：可点击重新拍摄

## 三、安全性评估

### 3.1 数据传输与存储加密 ⚠️

#### 现状评估

| 安全项 | 状态 | 说明 |
|--------|------|------|
| 本地存储加密 | ❌ 缺失 | SQLite 未启用 SQLCipher |
| 传输加密 | N/A | 本地模式不需要传输加密 |
| 文件加密 | ❌ 缺失 | 人脸照片、健康证照片未加密存储 |

| 优先级 | 措施 | 实施难度 |

| P1 | 使用 Android Keystore 存储密钥 | 中等 |
| P2 | 启用 SQLCipher 加密数据库 | 简单 |
| P3 | 照片文件加密存储 | 中等 |

// 建议：使用 EncryptedSharedPreferences
val encryptedPrefs = EncryptedSharedPreferences.create(

### 3.2 身份认证与权限控制 ⚠️

#### 现状评估

| 安全项 | 状态 | 说明 |
|--------|------|------|
| 登录机制 | ✅ 基础 | 账号+密码验证 |
| Token 管理 | ⚠️ 简单 | 内存存储，无过期机制 |
| 权限验证 | ⚠️ 部分 | 仅删除操作需要验证 |
| 会话管理 | ⚠️ 简单 | 无会话超时 |

#### 风险等级

| 风险 | 等级 | 说明 |
|------|------|------|
| Token 无过期 | 🟡 中等 | 长期有效，设备丢失后风险 |
| 权限验证不足 | 🟡 中等 | 创建/更新操作无需验证 |
| 密码弱加密 | 🔴 高 | 简单哈希存储，易被破解 |

#### 建议措施

| 优先级 | 措施 | 说明 |
|--------|------|------|
| P1 | 密码使用强哈希 | bcrypt 或 PBKDF2 |
| P2 | Token 添加过期时间 | 如 24 小时 |
| P3 | 关键操作增加权限验证 | 创建/更新也需要验证 |

```kotlin
// 建议：密码存储使用 bcrypt
import org.mindrot.jbcrypt.BCrypt

fun hashPassword(password: String): String {
    return BCrypt.hashpw(password, BCrypt.gensalt())
}

fun verifyPassword(password: String, hash: String): Boolean {
    return BCrypt.checkpw(password, hash)
}
```

### 3.3 输入验证与防注入措施 ⚠️

#### 现状评估

| 安全项 | 状态 | 说明 |
|--------|------|------|
| 参数校验 | ⚠️ 部分 | 接口设计有 ValidationError，但实现未确认 |

#### 风险等级

| 风险 | 等级 | 说明 |
|------|------|------|
| 路径遍历 | 🟡 中等 | 文件操作可能存在路径穿越 |
| 参数校验不足 | 🟡 中等 | 业务参数未全面校验 |

#### 建议措施

```kotlin
// 建议：参数校验工具类
object InputValidator {
    fun validateEmployeeId(id: String): Result<Unit> {
        if (id.isBlank()) return Result.failure(
            AppError.ValidationError("employeeId", "不能为空")
        )
        if (id.length > 20) return Result.failure(
            AppError.ValidationError("employeeId", "长度不能超过20")
        )
        if (!id.matches(Regex("^[A-Za-z0-9_]+$"))) return Result.failure(
            AppError.ValidationError("employeeId", "只能包含字母、数字、下划线")
        )
        return Result.success(Unit)
    }
    
    fun validateFilePath(path: String): Boolean {
        // 防止路径遍历攻击
        val normalized = File(path).canonicalPath
        return normalized.startsWith(ALLOWED_BASE_DIR)
    }
}
```
